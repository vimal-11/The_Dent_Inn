# The_Dent_Inn
A Neighbourhood Dental Clinic Website

## Tech Stack

- Python
- Django
- PostgreSQL
- Html/Css
- Heroku

## Features

- Appointment Bookings
- Online Consultation System
- Online Payment System (RazorPay Gateway API)
- OTP Authorization (Both Email and SMS)
